Minser is not your usual file conversion app.

It is aimed to be translated into the following languages:

* ربى
* Čeština
* Deutsch
* Ελληνικά
* English
* Español
* Français
* עִברִית
* "िंदी
* Italiano
* bahasa Indonesia
* 日本語
* Melayu
* Māori
* Norsk
* Português
* தமிழ்
* Українець
* 简体中文

A menubar/taskbar App based on <a href="http://electron.atom.io" target="_blank">Electron</a>.
